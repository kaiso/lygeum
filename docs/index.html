Configuration made easy

Lygeum is an open source centralized configuration server for modern applications, systems and services.
Lygeum comes with a complete toolbox allowing to create, manage and deploy application configurations quickly and reliably which makes it a great configuration management solution for microservices and complex cloud distributed deployments

Learn more

Easy to install and configure

Get started by running the official docker image, use the ergonomic and intuitive interface to configure environments, applications, clients...
With fine grained ACL permissions manage your users in a simple manner.

Rich integrated toolbox

Lygeum comes with different tools to be easily integrated in cloud environments and existing applications.
like the the Command Line Interface and many upcoming clients for different deployment scenarios.

Wide compatibility

Lygeum supports downloading configuration in different layouts like json, yaml and java properties which makes it compatible with most systems and applications.

Track configuration changes

Lygeum keeps tracking changes on configuration for each enviroment and application separately in a versionning system manner.

Fine grained ACL

Lygeum comes with an access control module which allows to manage user's roles and fine grained permissions. Service clients leverage also this feature
